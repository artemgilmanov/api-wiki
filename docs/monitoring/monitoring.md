### Monitoring
- Monitoring lets you know about:
  - Problems before/after they occure
  - What is going on with your API

What to Monitor:
- Request/Sec
- Number of failures
- Latency
- CPU
- Number of users
- Number of sessions
- Geographic destribution
- RAM
